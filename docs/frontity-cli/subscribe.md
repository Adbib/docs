# `subscribe`

Subscribe to Frontity newsletter.

```shell
npx frontity subscribe [options] [email]
```

## Arguments

### **`[options]`**

|            Option            | Description                                                                                                                                  |
| :--------------------------: | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `-h`, `--help`  | output usage information                                                                                                             |
### **`[email]`**

The _email_ where the Frontity newsletter will be sent
