# `subscribe`

Suscribe to Frontity newsleter.

```shell
npx frontity subscribe [options] [email]
```

## Arguments

### **`[options]`**

|            Option            | Description                                                                                                                                  |
| :--------------------------: | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `-h`, `--help`  | Output usage information                                                                                                             |
### **`[email]`**

The Frontity newsletter will be sent to this _email_
